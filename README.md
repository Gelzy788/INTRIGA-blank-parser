# INTRIGA PDF Parser 📄🗂️

Удобное десктопное приложение для автоматического парсинга PDF-бланков и генерации структурированных отчетов. Программа извлекает данные о посылках (штрихкоды, адреса, индексы, получателей) и мгновенно экспортирует их в форматы Excel и Word.

## 🚀 Установка и Запуск

Программа поставляется в виде удобного `.exe` установщика для Windows. Для работы **не требуется** предварительная установка Java — всё необходимое уже зашито внутри!

**[📥 Скачать последнюю версию (Установщик для Windows)](https://github.com/Gelzy788/INTRIGA-blank-parser/releases/tag/1.0)**

> **Как вставить ссылку:** После того как создашь релиз на GitHub (раздел Releases -> Draft a new release) и прикрепишь туда свой `.exe` файл, скопируй прямую ссылку на этот файл и вставь её вместо текста `https://github.com/Gelzy788/INTRIGA-blank-parser/releases/tag/1.0` в скобках.

### Как установить:
1. Скачайте `.exe` файл по ссылке выше.
2. Запустите установщик и следуйте инструкциям на экране.
3. Запустите приложение с ярлыка на Рабочем столе или из меню "Пуск".

## ✨ Основной функционал

* **Интеллектуальный парсинг PDF:** Извлечение текстовой информации и распознавание штрихкодов.
* **Экспорт в Excel (.xlsx):** Автоматическое формирование таблиц (стандарт Excel 2007+) с автоподбором ширины колонок.
* **Экспорт в Word (.docx):** Генерация готовых документов на основе шаблонов.
* **Кастомизация интерфейса:** Встроенная поддержка Темной и Светлой темы с сохранением пользовательских настроек.
* **Автономность:** Приложение работает полностью локально, без необходимости подключения к интернету.

---

## 🛠️ Технологический стек

* **Язык:** Java 17
* **Интерфейс:** JavaFX 21 (с использованием FXML и CSS)
* **Работа с документами:** Apache POI (Excel/Word), pdfbox (PDF), poi-tl
* **Генерация штрихкодов:** ZXing
* **Сборка и упаковка:** Maven (maven-shade-plugin), jpackage, WiX Toolset
